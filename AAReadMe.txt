DESCRIPTIF D'INTRODUCTION

Ce dossier contient 
- un rmd nommé TER.Rmd, 
- un fichier tex nommé TER.tex, 
- plusieurs tableaux Excel permettant la compilation du programme, 
- des images incorporées dans le RMD et enfin 
- le document TER.pdf


INSTALLATION
Pour que la compilation "knit to html" fonctionne, il est nécessaire d'installer préalablement les librairies suivantes: 
- stats
- agricolae
- leaps
- stringr
- readxl
- xtable
- dplyr



MISE EN MARCHE

Pour des soucis d'esthétique, nous avons choisi de ne pas effectuer l'affichage de tous les plots et les codes quand on Knit. 

On peut néanmoins les visualiser; il suffit de supprimer les "#" devant le code du RMD.
